# weather-api
Pet project with interaction with accuweather, will allow you to find out the weather in Zermatt.  
api/weather/current - Current temperature  
api/weather/historical - Hourly temperature for the last 24 hours )  
api/weather/historical/max - Maximum temperature for 24 hours   
api/weather/historical/min - Minimum temperature for 24 hours   
api/weather/historical/avg - Average temperature for 24 hours   
api/weather/by_time - Returns the weather for today or yesterday by hour (integer from 0 to 23) /health - Backend status (always respond with OK)
